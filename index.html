About

Macrosoft Paint (formerly Baintprush) is a simple raster graphics editor that has been included with all versions of MacOS. The program opens and saves files in Mac bitmap (BMP), JPEG, GIF, PNG, and single-page TIFF formats. The program can be in color mode or two-color black-and-white, but there is no grayscale mode. For its simplicity and that it is included with MacOS, it rapidly became one of the most used applications in the early versions of MacOS, introducing many to painting on a computer for the first time. It is still widely used for simple image manipulation tasks.
